# Agent Runbook

Use this runbook to understand the runtime flow.

1. Parse user query.
2. Stop for clarification if the region is unknown.
3. Plan tool calls.
4. Fetch weather fallback context.
5. Save memory context.
6. Search restaurant candidates.
7. Score and draft recommendations.
8. Fetch place details.
9. Run reflection checks.
10. Finalize the answer and submission trace.
