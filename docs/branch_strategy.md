# Branch and Commit Strategy

The assignment work is isolated on `assignment/food-agent-react-trace`.

## Strategy

- Keep `main` as the baseline AniPick branch.
- Add the assignment implementation as small, reviewable commits.
- Group commits by environment, data, parser, tools, agent, scripts, tests, and docs.
- Do not include generated zip files or real secrets in Git.

## Backup branch

`backup/assignment-food-agent-react-trace-before-split` preserves the first pushed version before history was split into smaller commits.
