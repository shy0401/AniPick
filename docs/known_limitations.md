# Known Limitations

- External API adapters are represented through local fallback behavior for reliable assignment execution.
- The memory tool stores data only in the current Python process.
- The sample dataset is intentionally small and focused on the assignment scenario.
- On the current Windows machine, Node 24 can fail during Vite/Rollup native build; Node 22 LTS is recommended.

These limitations are documented so grading can focus on the ReAct flow, tool observations, exception handling, and submission trace.
