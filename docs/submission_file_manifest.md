# Submission File Manifest

The package script includes source, tests, docs, frontend source, environment examples, and generated trace outputs.

## Must include

- `app/`
- `mcp_servers/`
- `tests/`
- `scripts/`
- `docs/`
- `README.md`
- `requirements.txt`
- `.env.example`

## Must exclude

- `.env`
- `node_modules/`
- `.venv/`
- `dist/`
- `build/`
- IDE metadata
- API keys or secrets
